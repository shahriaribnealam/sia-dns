# 🌍 SIA-DNS — Global DNS Propagation Checker

SIA-DNS is a free, fast, and reliable global DNS propagation checker. It allows developers, system administrators, and network engineers to instantly scan and verify DNS records across multiple servers worldwide, including dedicated nodes in Bangladesh.

🔗 **Live Website:** [https://sia-dns.pages.dev](https://sia-dns.pages.dev)

---

## ✨ Features

* **12+ DNS Record Types Supported:** Scan for `A`, `AAAA`, `CNAME`, `MX`, `TXT`, `NS`, `PTR`, `SOA`, `SRV`, `CAA`, `DS`, and `DNSKEY`.
* **Global Server Network:** Check DNS propagation across multiple continents, including regional testing from Bangladesh.
* **Instant Results:** Real-time feedback with a clean and modern user interface.
* **100% Free:** No hidden costs, accounts, or subscriptions required.

---

## 🛠️ Tech Stack

* **Frontend:** Vanilla HTML5, CSS3 (Custom Grid & Flexbox), and Native JavaScript (ES6+).
* **Mapping & UI:** Leaflet.js with CartoDB Light clean map tiles.
* **DNS Resolution:** Cloudflare DNS-over-HTTPS (DoH) JSON API.
* **Hosting:** Cloudflare Pages (Serverless & Fast Global CDN).
